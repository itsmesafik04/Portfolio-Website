<pre># Portfolio Website with Flask

A personal portfolio website built using Flask, HTML, and CSS to showcase professional information, technical skills, projects, and contact details. The website uses Flask routing and Jinja2 templates for dynamic page rendering and includes a contact form that allows visitors to submit messages.

## Features

* Responsive and user-friendly design
* Personal profile and introduction section
* Skills and projects showcase
* Contact form with Flask form handling
* Dynamic content rendering using Jinja2 templates

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* Jinja2

## Learning Outcomes

* Understanding Flask application structure
* Working with routes and templates
* Handling web forms and user input
* Organizing static files and templates
* Building a real-world web application using Python

## Project Outcome

This project demonstrates the fundamentals of web development with Flask, including routing, template rendering, form handling, and front-end integration.
